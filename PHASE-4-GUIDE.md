# NexCart — Phase 4: Reviews, Admin Panel, SEO & Performance

## Overview
**Duration:** 2 weeks (4 sprints)  
**Goal:** Review/rating system, admin moderation panel, SEO optimization with structured data, and performance audit for Lighthouse 95+.

---

## Sprint Breakdown

### Sprint 4.1 — Review System (Days 1-3)
- Reviews table with RLS (buyers who purchased can review)
- Star rating component
- Review form on product detail page
- Review list with pagination
- Aggregate rating updates on products/stores

### Sprint 4.2 — Admin Panel (Days 4-7)
- Admin layout with sidebar navigation
- Vendor approval/rejection workflow
- User management (role changes, suspensions)
- Product moderation (flag/remove)
- Platform overview dashboard with KPIs

### Sprint 4.3 — SEO & Metadata (Days 8-10)
- Dynamic metadata for all pages (generateMetadata)
- JSON-LD structured data (Product, Store, BreadcrumbList)
- Sitemap generation
- Open Graph images
- robots.txt

### Sprint 4.4 — Performance & Deploy (Days 11-14)
- Loading skeletons for all data pages
- Image optimization with next/image
- Code splitting verification
- Lighthouse audit + fixes
- Final Vercel production deploy
